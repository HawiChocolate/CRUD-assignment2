# Food Bank App
A modern Flutter mobile application for managing food bank inventory.

## About the App

**Food Bank App** helps food banks and charities efficiently manage their food inventory. Users can browse available items, add new donations, edit existing stock, delete items, search the inventory, and view detailed information.

## Technologies Used
- Flutter: (Dart)
- Bloc for state management
- Dio: for API calls
- GetIt: for dependency injection
- DummyJSON: as the backend API (`https://dummyjson.com/products/category/groceries`)

## API Note
This app uses DummyJSON (a free mock REST API). Any data you add, update, or delete will not persist permanently as it is a mock backend.
